# 波兰电影文化负载词汉译平行语料库

## 项目简介

本语料库是北京第二外国语学院“波兰语-汉语影视翻译文化负载词研究”项目的核心成果。聚焦于波兰电影字幕中**文化负载词**的翻译处理，通过人工识别与标注，构建了一个“小而精”的双语平行语料库，旨在为波兰语学习者、翻译从业者及跨文化研究者提供可查询、可参考的“文化词典”式资源。

## 语料库概况

| 项目 | 说明 |
|------|------|
| **语料规模** | 85条文化负载词条目 |
| **电影来源** | 《盗走达芬奇》(VINCI) 与《冷战》(Zimna wojna) |
| **标注维度** | 文化分类（5类）、翻译策略（6类）、上下文语境、翻译得失分析 |
| **数据格式** | 结构化表格 + 动态网页展示 |

## 选题价值

### 学术价值
填补波兰语-汉语影视翻译微观研究的空白，为文化翻译理论（如Venuti归化/异化策略、Aixela文化专有项分类）提供实证案例，探索中东欧语言文化负载词在中文语境下的转换规律。

### 应用价值
- **学习者**：直观了解波兰文化特色词汇的译法及背后的文化逻辑
- **翻译从业者**：查阅同类文化词的翻译策略选择，获取参考案例
- **跨文化创作者**：理解波兰文化符号的中文表达方式

## 语料库构建流程

### 1. 选片与采集
选取在文化代表性上具有多样性的波兰电影：
- **《盗走达芬奇》**（VINCI，2004）：犯罪喜剧片，涉及艺术、历史、社会阶层等多维文化元素
- **《冷战》**（Zimna wojna，2018）：历史爱情片，涉及政治体制、宗教、民俗等深层文化内涵

获取波中双语字幕文本，进行基础清洗（去除时间码，保留纯对话）。

### 2. 对齐与初建
使用简单工具结合人工校对，确保波中文本实现精确的**句级对齐**，形成基础平行语料。

### 3. 核心环节：人工识别与标注（项目核心）
依据文化负载词理论框架，小组成员共同阅读平行文本，完成以下工作：

#### 识别
在波兰语文本中划出所有体现独特文化的词、短语、典故等。

#### 标注维度
| 字段 | 说明 |
|------|------|
| **ID** | 唯一标识符（如VINCI-001） |
| **电影名称** | 来源电影 |
| **时间戳** | 对白在字幕文件中的时间位置 |
| **原文上下文** | 包含文化负载词的完整波兰语句子 |
| **目标文上下文** | 对应的中文翻译句子 |
| **文化负载词(PL)** | 识别出的波兰语文化词 |
| **对应译文(CN)** | 中文译文 |
| **文化分类** | 物质文化、社会文化、观念文化、语言文化、生态文化（五选一） |
| **翻译策略** | 直译、意译、替代、音译、加注、省略（可选1-2项） |
| **备注/分析** | 简析翻译得失、文化内涵损失或补充信息 |

#### 质量控制
通过**多人交叉校验**确保标注的一致性与准确性。

### 4. 入库与呈现
- **数据存储**：结构化表格（Excel）+ 轻量级数据库
- **成果展示**：**动态静态网页**（本仓库），支持多维度筛选、搜索与排序

## 创新点

### 视角创新
从“文化难点”入手，不同于普通的全文字幕库，本项目直接聚焦翻译中最棘手、最具研究价值的“文化负载词”，使语料库建设由“面”聚焦到“点”，**深度而非广度取胜**。

### 方法创新
通过系统性的识别、分类与对照，构建一个高质量、高附加值的小型专项语料库。这种“**小而精**”的策略在入门阶段更具可行性，为后续扩展奠定方法论基础。

### 应用创新
创建“**翻译策略案例库**”：不仅记录词汇对照，更分析并归纳每个案例的翻译策略，使得本语料库不仅能查询“怎么译”，还能理解“为何这样译”，成为生动的**教学与研究案例库**。
